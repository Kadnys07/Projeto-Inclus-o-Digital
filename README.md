# Conecta+

Resumo
-------
Conecta+ é um projeto simples de inclusão digital com páginas estáticas em HTML e CSS. O objetivo é oferecer uma interface leve com tutoriais, iniciativas e uma página de login para apoiar pessoas que estão começando a usar a internet e ferramentas digitais.

Recursos
--------
- Página inicial com chamadas para tutoriais e iniciativas.
- Galeria de tutoriais com cartões que apontam para vídeos externos.
- Página de iniciativas com links para cursos e programas relevantes.
- Página de login com formulário (implementação front-end apenas).

Estrutura do projeto
--------------------
- `index.html` — Página inicial.
- `tutoriais.html` — Galeria de tutoriais (cards com botões).
- `iniciativas.html` — Lista de iniciativas com botões de acesso.
- `login.html` — Formulário de login (cliente).
- `style.css` — Estilos e layout do site.
- `script.js` — Comportamento (se presente para login ou interações).

Como executar
-------------
1. Abra `index.html` no seu navegador (duplo clique) ou sirva a pasta com um servidor HTTP local.
2. Edite os arquivos HTML/CSS conforme necessário para personalizar conteúdo e estilos.

Tecnologias
-----------
- HTML5
- CSS3 (Flexbox)
- Bootstrap (CDN usado para reset/estilos básicos)

Contribuição

Projeto criado como exemplo educativo para inclusão digital.
